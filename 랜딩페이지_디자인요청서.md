## 0. 한 줄 요약

**"수출 노하우 없는 소규모 K뷰티 브랜드 대표가, 유럽 바이어의 답장을 받고 3개월 안에 첫 계약 테이블에 앉게 한다"** — 이 약속을 방문자가 10초 안에 "내 얘기다"라고 느끼게 만드는 랜딩페이지.

---

## 1. 프로젝트 배경

- **서비스명:** BridgeX (Bridge + X)
- **무엇인가:** K뷰티 소규모 브랜드의 해외 수출 실무를 대신 실행하는 **Human + AI 하이브리드 수출 대행 플랫폼**
- **핵심 차별점:** "있어 보이는 컨설팅 보고서"가 아니라 **지금 바로 보낼 수 있는 이메일 한 통·대응 스크립트 하나**를 만들어 준다. AI가 반복 실행을, 글로벌 실무 전문가가 협상·전략을 맡는다.
- **현재 상태:** 1차 랜딩 초안(`bridgex_landing.html`)이 이미 존재함. 이 요청서는 **그 초안을 기준으로 완성도를 끌어올리는 것**이 목표. 톤·구조는 유지하되, 아래 요구사항을 반영해 다듬어 주세요.

---

## 2. 이 페이지의 유일한 목표 (Primary Goal)

**무료 상담 신청(이메일 리드 확보).**

- 모든 섹션은 이 하나의 전환으로 수렴해야 함. 다른 CTA로 방문자를 분산시키지 말 것.
- 서브 목표: "서비스 알아보기"로 스크롤 유도 (아직 신뢰가 안 쌓인 방문자용).

## 3. 타깃 고객 (누구에게 말하는가)

| 항목 | 내용 |
|---|---|
| 대표 페르소나 | 40대 초, 바디케어 제품을 갓 출시한 **1인 소규모 화장품 창업자** (예: 에코라랩) |
| 상황 | 유럽 수출이 목표지만 수출 프로세스 전반에 노하우가 없음 |
| 현재 하는 것 | 국내 박람회 참가 + 한글→AI번역 콜드 이메일 + 지인 소개 |
| 가장 아픈 지점 | **이메일 10통 발송 / 긍정 답변 0건 / 유럽 바이어 접점 0건.** 번역은 되는데 "바이어가 반응하는 방식"을 모름 |
| 감정 | 혼자 다 하다 지침. 컨설팅은 비싸서 엄두도 못 냄 |

> **디자인 함의:** 방문자는 "대기업 마케터"가 아니라 **지치고 예산 없는 1인 창업 대표**다. 화려함보다 "이건 내 문제를 정확히 안다"는 **정밀한 공감**이 신뢰를 만든다.

## 4. 핵심 메시지 & 카피 (확정본)

### 메인 헤드라인
> **혼자 감당하던 해외 수출, 이제 BridgeX가 끝냅니다.**

### 서브헤드
> 바이어 찾기부터 계약 직전까지 — K뷰티 수출의 모든 실행을 대신합니다.
> 대표님은 제품에만 집중하십시오.

### 브랜드 한마디
> "혼자 다 하다 지친 K뷰티 창업자에게, 해외 수출만큼은 전부 맡길 수 있는 파트너."

### 가치제안 한 문장
> 수출 노하우 없는 소규모 K뷰티 브랜드가 바이어 발굴·이메일·규제·협상을 혼자 해결하지 않고도, 3개월 안에 유럽 바이어와 계약 테이블에 앉게 한다.

*(카피는 이미 초안에 반영되어 있음. 임의로 바꾸지 말고, 문구 다듬기가 필요하면 의도를 유지한 선에서만.)*

## 5. 페이지 구조 (섹션 순서 = 설득 흐름)

방문자를 **문제 인식 → 해법 이해 → 차별성 확신 → 변화 상상 → 신뢰 → 행동** 순서로 끌고 간다.

1. **NAV** — 로고 `BridgeX` + 우측 CTA "무료 상담 신청"
2. **HERO** — 헤드라인 + 서브헤드 + 이중 CTA(주: 상담신청 / 부: 서비스 알아보기) + 신뢰 지표 3개(3개월 / 30분 / 100% K뷰티 특화)
3. **PROBLEM** — "바이어를 만나도 계약까지 못 가는 이유" + **수출 7단계 중 Stage 4~5(제안·협상/계약)가 '데스밸리'** 임을 시각화 + 페인포인트 3개
4. **HOW IT WORKS** — 6단계 실행 프로세스. 각 단계에 **[AI 자동화] / [전문가 직접 개입]** 태그로 하이브리드 구조를 명확히
5. **COMPARISON** — "지금 혼자 할 때 vs BridgeX" 6행 비교표
6. **BEFORE / AFTER** — "3개월 전과 후" 좌우 카드 대비
7. **VALUES** — 타협하지 않는 3원칙(실행 가능성 / 소규모 접근성 / K뷰티 특화 깊이) + 각 "증명 행동"
8. **CTA** — 이메일 입력 + "무료 상담 신청" + 안심 문구(스팸 없음·언제든 취소·1영업일 내 연락)
9. **FOOTER** — 로고 + 카피라이트

> 섹션을 임의로 추가/삭제하지 말 것. 순서가 곧 설득 논리임.

## 6. 비주얼 디렉션 (브랜드 룩앤필) — ⚠️ 라이트 테마로 전면 전환

> **중요:** 기존 초안은 다크 네이비였으나, **밝고 깨끗한 라이트 테마로 새로 간다.** 화장품 브랜드다운 **따뜻하고 정갈한 프리미엄** 느낌으로. 어둡고 무거운 인상 금지.

### 톤
**신뢰 · 프리미엄 · 정갈함 · 여백.** "작은 회사지만 글로벌 실무 수준"이라는 인상. 밝은 배경 위에 골드 포인트로 고급감을 낸다. K뷰티 스킨케어 브랜드처럼 **깨끗하고 숨 쉬는 화면**.

### 컬러 (라이트 팔레트로 교체)
| 역할 | 값 |
|---|---|
| 배경(메인) | `#FAF8F4` 웜 오프화이트(크림) |
| 배경(섹션 교차) | `#F2EDE4` 연한 베이지 |
| 카드 | `#FFFFFF` 화이트 (은은한 그림자 `0 2px 20px rgba(30,25,15,0.06)`) |
| 포인트(브랜드) | `#B8912E` 딥 골드 (텍스트·아이콘용, 대비 확보) / `#C9A84C` 골드 (버튼·강조 채움용) |
| 본문 텍스트 | `#22252E` 차콜(거의 검정) |
| 제목 텍스트 | `#141824` 딥 잉크 |
| 보조 텍스트 | `#6B7280` 그레이 |
| 위험/문제 강조 | `#D64545` 레드 (연한 배경 `#FBEBEB`) |
| 성공/해결 강조 | `#2E9E6B` 그린 (연한 배경 `#E9F5EF`) |

→ **크림 배경 + 차콜 텍스트 + 골드 포인트.** 골드는 CTA·강조에만 아껴 쓰고 남발 금지. 흰 카드는 은은한 그림자로 띄운다(테두리보다 그림자로 구분).

### 타이포그래피
- 영문/숫자: `Inter` (헤드라인 700~800 굵기, letter-spacing 살짝 조임)
- 한글: `Noto Sans KR`
- 위계 뚜렷하게, 본문 `line-height 1.7~1.8`. 밝은 배경이므로 폰트를 너무 얇게(300) 쓰지 말 것 — 최소 400.

### 디자인 원칙
- **여백을 크게.** 정보 밀도를 낮추고 화면이 숨 쉬게 — 이게 이번 개편의 핵심.
- 문제 섹션은 연한 레드 톤으로 살짝 긴장을, 해결 섹션은 화이트/골드로 안도를. 다크 대비가 아니라 **밝은 톤 안에서의 미묘한 대비**.
- 은은한 인터랙션(호버 시 카드 살짝 떠오름/그림자 진해짐, 스크롤 진입 fade-up)은 좋되 과하지 않게.
- 아이콘은 절제. 이모지보다 **심플한 라인 아이콘**이 라이트 프리미엄 톤에 더 맞음.

## 6-1. 텍스트 다이어트 (⚠️ 이번 개편 최우선)

> 기존 초안은 **글이 너무 많다.** 방문자는 읽지 않고 훑는다. 아래 원칙으로 카피를 대폭 덜어낼 것.

- **각 섹션 본문은 1~2문장으로.** 설명 문단을 짧은 리드 카피 + 핵심 키워드로 압축.
- **HOW IT WORKS 6단계** → 각 단계 설명을 **한 줄(최대 두 줄)**로. 지금처럼 3~4줄 설명 금지.
- **페인포인트/밸류 카드** → 제목은 강하게, 설명은 한 줄로.
- 비교표는 각 셀을 **짧은 구절**로 (문장 X). 예: "AI가 국가별 DB 자동 매칭".
- "말로 다 설명"하지 말고 **레이아웃·아이콘·굵기·색으로** 의미를 전달. 글은 거들 뿐.
- 핵심 메시지(4번 헤드라인·서브헤드·가치제안)는 유지하되, 그 외 부연 설명은 과감히 삭제·압축.

**목표 감각:** 스크롤하면서 "읽는다"가 아니라 "훑고도 이해된다"가 되도록.

## 7. 기술 요구사항

- **단일 HTML 파일**, CSS는 `<style>` 인라인 (외부 의존성 최소화). 폰트만 Google Fonts 허용.
- **완전 반응형.** 특히 모바일(≤768px)에서 2단 그리드(문제/비교/스텝/밸류)가 1단으로 자연스럽게 접혀야 함. Before/After 화살표는 세로 회전.
- 비교표는 모바일에서 가로 스크롤 없이 읽히도록 폰트·패딩 축소 대응.
- CTA 이메일 폼: 지금은 프론트 UI만 (실제 전송 연동은 추후). `type="email"` + placeholder 유지.
- 접근성 기본: 충분한 명도 대비, 시맨틱 태그, 포커스 상태 표시.
- 부드러운 스크롤 앵커(`#how`, `#cta`) 동작.

## 8. 하지 말아야 할 것 (Guardrails)

- ❌ "혁신·최고·No.1" 같은 근거 없는 과장 카피 추가 금지 (실행 가능성·정직함이 브랜드 코어)
- ❌ 아직 없는 수치(고객 수, 성공 사례 N건, 후기)를 지어내 신뢰 배지로 넣지 말 것 — **없는 실적을 꾸미지 않는다**
- ❌ 여러 CTA로 목표 분산 금지 (오직 "무료 상담 신청")
- ❌ K뷰티 외 타 산업으로 확장하는 듯한 표현 금지 (특화 깊이가 차별점)
- ❌ 팝하고 캐주얼한 스타트업 톤으로 변질 금지 — 프리미엄·신뢰 유지
- ❌ 다크 배경 사용 금지 — 라이트(크림 오프화이트) 테마로 통일
- ❌ 긴 설명 문단 금지 — 섹션당 1~2문장, 나머지는 레이아웃·아이콘으로
